Runtime Bridge Layer Demo

The JavaScript engine loads bridge frameworks at runtime from reedstyle.bridge.yaml

How it works:

  1. User defines frameworks in reedstyle.bridge.yaml
  2. JavaScript reads the YAML at runtime
  3. Creates a <style> tag with @layer bridge
  4. Adds @import statements for enabled frameworks
  5. Inserts it BEFORE the ReedSTYLE CSS link

✅ Runtime Loading

User NEVER runs build process - just uses:

  • dist/reedstyle.css
  • dist/reedstyle.js
  • YAML configuration files
